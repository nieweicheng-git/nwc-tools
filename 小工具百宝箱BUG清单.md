## 一、整体页面基础 BUG

1. **背景样式完全不符合规范**
问题：未使用设计要求的 135° 柔和渐变背景，为单调浅灰纯色，无分层、无毛玻璃、无深度感
违反规范：需背景层 / 容器层 / 内容层三层视觉
修复要求：使用指定浅色渐变背景，实现毛玻璃通透深度感

2. **页面边距与布局错误**
问题：页面内边距≠上下 40px / 左右 24px，导航栏贴顶，大屏无标准留白、未居中锁定 1200px
违反规范：PC 端固定最大宽度、标准内边距、居中布局
修复要求：内容居中、留白规范、不贴边、不撑满

3. **顶部提示文本未高亮**
问题：「展示 20/20 个」文字为普通灰色，无高亮、无强调样式，不醒目
违反规范：需突出显示工具数量状态
修复要求：保留该文字，改为**主题色加粗高亮**样式，保持整洁且突出

## 二、搜索框 BUG（核心重点）

1. **搜索框边框异常错乱**
问题：搜索框出现多层彩色错位边框（左侧红、中间蓝、右侧紫），不是设计要求的统一主题色边框，样式极度混乱
违反规范：搜索框边框需统一主题色，聚焦时仅显示柔和高亮边框，无多层错位
修复要求：删除多余的调试边框，统一为主题色边框，聚焦时仅显示一层柔和高亮

2. **英文调试提示泄露（最高优先级）**
问题：搜索框右侧出现英文调试提示「Press Enter to apply」，生产环境暴露开发调试提示，不符合中文界面规范；且输入框未完全填满，输入文字时该提示仍显示
违反规范：生产环境禁止显示任何开发调试信息，所有提示需为中文；输入框需按设计规范填充，无冗余调试提示
修复要求：**彻底删除该英文调试提示**，输入框样式调整为设计要求的填充状态，如需保留交互提示可改为中文「按回车搜索」

3. **搜索框尺寸 / 样式错误**
问题：宽≠400px、高≠40px、圆角≠20px，输入文字「腾讯」未居中、靠左对齐，无毛玻璃背景
违反规范：搜索框尺寸、文字居中、毛玻璃背景标准化
修复要求：尺寸标准、文字居中、毛玻璃背景，与导航栏风格统一

## 四、工具卡片 BUG（最高优先级）

1. **卡片元素严重溢出 / 错位（最高优先级）**
问题：Sourcetree 卡片功能标签「GUI」、Regex101/JSON 格式化 / Diffchecker 卡片的操作按钮，全部跑到卡片**外部**，溢出卡片边界、错位
违反规范：所有卡片元素必须在卡片内部，不允许溢出、不允许错位
修复要求：将所有溢出的标签、按钮全部移入卡片内部，对齐整齐，无任何元素溢出卡片

2. **卡片结构严重割裂（重点修复）**
问题：使用 / 删除按钮全部放在卡片**外部**，形成 “卡片一行、按钮一行”，布局破碎，部分卡片按钮直接溢出卡片
违反规范：所有元素必须在卡片内部，卡片为完整整体
修复要求：按钮全部移入卡片内，底部居中，卡片完整统一

3. **卡片非全域点击**
问题：仅按钮可点，卡片主体无点击效果，体验极差
违反规范：整个卡片为点击区域
修复要求：点击卡片任意位置均可跳转 / 下载

4. **来源标签样式错误**
问题：圆角≠8px，字体≠11px，内边距不规范，无样式区分
违反规范：右上角小标签、标准尺寸、居中
修复要求：来源标签样式统一、位置正确

5. **删除按钮默认显示**
问题：删除按钮一直显示，堆在卡片外，页面杂乱、易误触
违反规范：仅悬停卡片时显示，位于右下角
修复要求：默认隐藏，悬停显示，位置规范

6. **卡片无悬停动效**
问题：无上浮 4px、无阴影加深、无 0\.25s 过渡
违反规范：悬停有上浮动效与阴影强化
修复要求：悬停卡片上浮、过渡丝滑

7. **自定义添加工具显示异常（重点修复）**
问题：自己添加的工具（如 JSON 格式化），卡片样式错乱、按钮溢出、UI 不统一，且会泄露多余的 HTML 标签（如`\&lt;/div\&gt;`、`\&lt;div class=\&\#34;card\-divider\&\#34;\&gt;\&lt;/div\&gt;`），导致页面显示异常代码
违反规范：所有卡片样式必须完全统一，禁止泄露任何 HTML 代码
修复要求：自定义工具与系统工具样式完全一致、无异常，彻底清除泄露的 HTML 标签代码

## 五、分页功能 BUG

1. **分页控件错位 / 样式错误**
问题：分页按钮错位，选中的页码 1 为纯色红色，与其他按钮不对齐，无渐变高亮，与页面风格不匹配
违反规范：分页按钮需对齐整齐，选中态使用渐变高亮，样式统一
修复要求：分页按钮对齐整齐，选中态渐变高亮，与页面风格统一

2. **分页与搜索不联动**
问题：搜索时分页未同步更新，仍显示 2 页，未根据搜索结果调整分页
违反规范：搜索结果需同步更新分页状态，无结果时隐藏分页
修复要求：搜索时同步更新分页，无结果时隐藏分页控件

3. **分页切换无动画**
问题：分页切换时，工具卡片直接生硬刷新、跳变，无任何平滑过渡
违反规范：分页切换时，工具卡片需伴随 0\.3s 淡入淡出动画
修复要求：添加分页切换动画，切换时工具卡片平滑淡入淡出

4. **分页控件过大不和谐**
问题：分页按钮尺寸过大，与页面整体风格不统一，视觉突兀
违反规范：分页控件需小巧精致，与页面元素尺寸匹配
修复要求：缩小分页按钮尺寸，调整为与页面风格匹配的小巧样式
